# VTM Chronicle Manager

Структурированная система ведения хроники **Vampire: The Masquerade 20th Anniversary Edition** в связке с Claude Code как Рассказчиком и сценаристом.

Работает с любым городом, любой эпохой, любым составом кланов и сект. Настройка под домен — один файл (`CLAUDE.md`), всё остальное готово к работе.

---

## Быстрый старт

**1.** Запусти мастер настройки домена:

```powershell
.\tools\new_city.ps1
```

Введи город, год и (опционально) районы — скрипт заменит все плейсхолдеры в файлах и создаст папки локаций.

**2.** Заполни секции `[ЗАПОЛНИТЬ]` в `CLAUDE.md` (политика, история, локации).

**3.** Запусти первые инструменты:

```powershell
.\tools\new_npc.ps1 -Name "Имя" -Type vampire   # первый персонаж
.\tools\new_module.ps1                           # первый модуль
```

**4.** Подробная инструкция — `SETUP.md`.

## Справочник команд

| Задача | Команда |
|---|---|
| **Настройка нового домена** | `.\tools\new_city.ps1` |
| Новый канонический НПС | `.\tools\new_npc.ps1 -Name "Имя" -Type vampire` |
| Новый модульный НПС | Claude создаёт по Шаблону Г в `modules/.../нпс/` |
| Продвинуть НПС в канон | `.\tools\promote_npc.ps1 -ModuleName "..." -NpcName "Имя"` |
| Новый модуль | `.\tools\new_module.ps1` |
| Новая локация | `.\tools\new_location.ps1 -Name "..." -District "Район" -Zone neutral` |
| Дневниковая запись | `.\tools\add_diary.ps1 -CharacterName "Имя" -Period "ГГГГ-ММ"` |
| Записать сессию | Описать события → Claude заполнит хронику |
| Написать сцену | `/literary-writer` + описание |
| Проверить ссылки | `.\tools\validate_links.ps1` |
| Статусы персонажей | `.\tools\status.ps1` |
| Поиск по проекту | `.\tools\search.ps1 "запрос"` |
| Чеклист | `rules/CHECKLIST.md` |

## I. Персонажи

Полный протокол — `rules/npcs_city.md`. Чеклист — `rules/CHECKLIST.md` → секция 1.

### Канонические vs. модульные НПС

| Тип | Где хранится | Шаблон | Как создать |
|---|---|---|---|
| **Канонический** | `characters/[линейка]/[Имя]/` | А (вампир) / Б (смертный) / В (фея) | `.\tools\new_npc.ps1` |
| **Модульный** | `modules/[модуль]/нпс/[Имя]/` | Г (упрощённый) | Claude по запросу |

**Продвижение в канон:** НПС выжил + упомянут в `финал.md` + появился в 2+ модулях — или явный запрос ГМ. Инструмент: `.\tools\promote_npc.ps1`.

### V20-листы

Создаются в `[Имя]-лист.md` рядом с карточкой. Правила — `rules/character_sheet_v20.md`. По запросу для ПК и ключевых НПС-бойцов.

### Переименование

Требует правок в 10+ файлах. Всегда: `rules/CHECKLIST.md` → секция 3, затем `.\tools\validate_links.ps1`.

## II. Модули

Полный протокол — `rules/module_rules.md`. Чеклист — `rules/CHECKLIST.md` → секция 2.

```powershell
.\tools\new_module.ps1
# Интерактивный визард: связь с хроникой, список НПС и локаций
```

После создания:
1. Заполнить `[описание].md` и `сценарий.md`
2. Заполнить `нпс.md`, создать карточки Шаблон Г в `нпс/`
3. После сессии — добавить запись в хронику по протоколу `rules/chronicle.md`
4. Обновить `rules/open_threads.md`
5. Создать дневники через `.\tools\add_diary.ps1`

## III. Дневники

```powershell
.\tools\add_diary.ps1 -CharacterName "Имя" -Period "ГГГГ-ММ"
.\tools\add_diary.ps1 -CharacterName "Имя" -Period retrospective
.\tools\add_diary.ps1 -CharacterName "Имя" -Period "ГГГГ-ММ" -ModuleRef "название_модуля"
```

Хранятся в `characters/[линейка]/[Имя]/Journal_[Имя]/`. Правила — `rules/diary_rules.md`.

---

## IV. Локации

```powershell
.\tools\new_location.ps1 -Name "Название" -District "Район" -Zone dangerous
.\tools\new_location.ps1 -Name "Название" -District Другие -Module "модуль" -Zone neutral
```

Доступные зоны: `safe` · `neutral` · `dangerous`. Углы: `wide-angle` · `street-level` · `low angle` · `corridor`.

## V. Что автоматизировано, что остаётся Claude

| Инструмент | Автоматически | Требует Claude |
|---|---|---|
| `new_npc.ps1` | Папка + шаблон карточки + индексы | Биография, голос, промт изображения |
| `promote_npc.ps1` | Перемещение + обновление всех ссылок | Расширить биографию, заполнить иерархию |
| `new_module.ps1` | Папка + все файлы-заготовки | Сценарий, сцены, кульминация |
| `new_location.ps1` | Папка + шаблон + базовый промт | Атмосфера, сенсорная палитра, крючки |
| `add_diary.ps1` | Папка журнала + шаблон записи | Текст записи, зеркальные ссылки |
