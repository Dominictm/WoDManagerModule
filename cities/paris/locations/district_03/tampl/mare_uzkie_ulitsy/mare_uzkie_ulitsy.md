# Марэ — Узкие Улицы и Особняки, 3-й округ

> **Название:** Исторический квартал | **Округ:** 3-й | **Район:** Марэ | **Адрес:** Rue des Archives / Rue de Bretagne | **Зона:** 🟡 Серая (нейтральная) | **Контроль:** Примогены (резиденции), улицы без хозяина

---

## 🎭 Атмосфера

Средневековые переулки, уцелевшие вопреки Осману. Плющ тянется по камню, фонарь отбрасывает зелёный свет сквозь листву — и тот, кто стоит в тени арки, видит вас раньше, чем вы слышите шаги. Здесь живут самые старые особняки Парижа — *hôtels particuliers*, — и в некоторых из них владелец не менялся с XVII века. Марэ не спрашивает, кто ты. Марэ просто запоминает.

## 👁️ Сенсорная палитра

| Канал | |
|---|---|
| **Свет** | Единственный фонарь в конце переулка; зелёный свет сквозь плющ; золотое окно в стене — кто-то не спит |
| **Звук** | Собственные шаги по булыжнику; далёкий звук джазового клуба; капли с листьев; в туман — эхо удваивает шаги |
| **Запах** | Мокрый камень, старая штукатурка, плющ и сырость, иногда — запах дыма из невидимого камина |
| **Тактильное** | Шероховатость стен из белого парижского камня; брусчатка скользкая после дождя; арки создают температурные карманы |

## 🩸 Контекст Камарильи / Масок

| | |
|---|---|
| **Статус** | Жилая зона Примогенов и состоятельных анцилл; улицы нейтральны, частные особняки — личная территория |
| **Фракция** | Тореадор (арт-резиденции); Вентру (несколько особняков как инвестиции); Тремер (один особняк с заложенным кирпичом окном) |
| **Постоянные фигуры** | Смертные туристы поздним вечером (ЛГБТ-квартал активен до полуночи); консьержи-гули в воротах; изредка — Носферату в сточных тоннелях под улицей |
| **Угрозы** | Нарушение частной территории Примогена без приглашения = политический инцидент; туристы с телефонами создают риск Маскарада |
| **Маскарад** | 🟡 Средний — туристы до 23:00, после — почти пусто |

## 🪝 Сценарные крючки

1. Дверь одного из hôtels particuliers открыта настежь уже третью ночь. Свет внутри горит. Никто из соседей-Каин не знает, кому принадлежит особняк.
2. Тремер просит передать конверт по адресу на rue des Francs-Bourgeois — но не звонить, не стучать, просто оставить под дверью в 2:17 ночи.
3. Молодой художник-смертной работает в мастерской до рассвета, и его последние холсты показывают лица, которые он не мог видеть.
