# Лувр — Закрытые Галереи, 1-й округ

> **Название:** Музей (закрытые галереи) | **Округ:** 1-й | **Район:** Лувр | **Адрес:** Rue de Rivoli | **Зона:** 🏛️ Элизиум (для старейшин) | **Контроль:** Тореадор / ограниченный доступ

---

## 🎭 Атмосфера

После полуночи бронзовые замки поворачиваются ключами, которых нет ни в одном официальном реестре. Галереи погружаются в тишину — только аварийное освещение оставляет длинные полосы света на паркете. Картины смотрят. Те, кто бывал здесь достаточно долго, уверяют: особенно после трёх ночи некоторые портреты меняют выражение лица. Лувр — не музей. Это хранилище памяти тех, кто пережил всё, что на этих холстах изображено.

## 👁️ Сенсорная палитра

| Канал | |
|---|---|
| **Свет** | Аварийные зелёные указатели вдоль плинтусов; редкие дежурные лампы над самыми ценными полотнами; абсолютная темнота в боковых залах |
| **Звук** | Собственные шаги — единственный звук; далёкий гул системы климат-контроля; иногда — едва слышный звон со стороны стеклянной пирамиды |
| **Запах** | Лак, старый холст, пыль веков, реставрационный растворитель; в восточном крыле — устойчивый запах озона (здесь работал Тремер) |
| **Тактильное** | Прохлада музейного воздуха (+18°C постоянно); скользкий паркет в темноте; ощущение, что за плечом кто-то стоит |

## 🩸 Контекст Камарильи / Масок

| | |
|---|---|
| **Статус** | Неофициальный Элизиум для закрытых приёмов старейшин; открытых встреч здесь не анонсируют |
| **Фракция** | Тореадор (коллекционный интерес); Тремер (хранят неизвестный артефакт в подвале Ришелье); Джованни (периодически появляются — древние предметы их интересуют) |
| **Постоянные фигуры** | 3 гуля-охранника с ночными пропусками; таинственный реставратор-Тореадор, работающий без света |
| **Угрозы** | Камеры наблюдения — некоторые из них работают на нестандартной частоте; ловушки Тремера в секции египетских артефактов; Джованни не всегда приходят одни |
| **Маскарад** | 🟢 Низкий (ночью) → 🔴 Высокий (после 7 утра — первые сотрудники) |

## 🪝 Сценарные крючки

1. Полотно из закрытого хранения — портрет 1847 года — исчезло. На месте — отпечаток ладони в пыли, нечеловеческого размера.
2. Тремер просит передать послание реставратору-Тореадору. Послание — запечатанный свиток, и Тремер специально оговаривает: не читать.
3. Джованни заявили, что один из залов построен над старым некрополем и хотят провести «инспекцию» — без ведома Камарильи.
