# Барбес / Гут-д'Ор — Живой Город, 18-й округ

> **Название:** Жилой квартал / торговая зона | **Округ:** 18-й | **Район:** Гут-д'Ор | **Адрес:** Boulevard Barbès | **Зона:** 🔴 Серая / Анархи | **Контроль:** Сеттиты (ночные клубы), Анархи (улицы)

---

## 🎭 Атмосфера

Арабские вывески и неоновый знак «BOUCHERIE HALAL» — это не экзотика, это Париж как он есть. Улица живёт до 3 ночи: «Épicerie Orientale» открыта, музыка из второго этажа, запах специй и сырого мяса. Для Камарильи это периферия — для Сеттитов это рынок. Каждый второй смертный здесь что-то продаёт или чего-то хочет. Это делает место идеальным для охоты и идеальным для того, чтобы исчезнуть.

| Канал | |

| **Запах** | Специи (тмин, кориандр), жареное мясо из шаурмы, дождь на асфальте, кальян из глубины кафе |
| **Тактильное** | Толчея даже поздно ночью; влажный асфальт после дождя; ощущение, что за тобой следят — и это не паранойя |

## 🩸 Контекст Камарильи / Масок

| | |
|---|---|
| **Статус** | Серая зона; Камарилья формально претендует, фактически не патрулирует; Анархи считают своим |
| **Фракция** | Сеттиты (сеть ночных клубов и наркоторговли); Анархи иммигрантского происхождения (идеологизированы, связаны с реальной политикой бьёу); Носферату (информаторы среди бездомных) |
| **Постоянные фигуры** | Торговец специями-Сеттит (front Épicerie); анархи-Бруха из Алжира и Мали; гуль-охранник в клубе на rue de la Goutte d'Or |
| **Угрозы** | Полицейские рейды по ночам после 2010 (после беспорядков 2005); Сеттиты предлагают сделки, которые звучат выгодно; Анархи не приветствуют Камарилью |
| **Маскарад** | 🟡 Средний — много людей, мало камер; телефоны у всех |

## 🪝 Сценарные крючки

1. Сеттит предлагает информацию о Шабаше в обмен на «маленькую услугу»: доставить пакет на rue Marx Dormoy без вопросов. Пакет тёплый.
2. Молодой Анарх-Бруха хочет поговорить. Он знает кое-что о Примогене Вентру — и готов продать. Цена — защита его «семьи» в бьёу.
3. В 2 ночи épicerie закрылась. В витрине — надпись на крови, на арабском. Единственный, кто умеет читать арабский в котери, говорит: «Он уже здесь».
