# SETUP — Настройка нового домена VtM

Этот шаблон — готовый фреймворк для ведения Vampire: The Masquerade 20th Anniversary Edition хроники с помощью Claude Code.

---

## Шаг 1 — Заполни CLAUDE.md

Открой `CLAUDE.md` и замени все `[ЗАПОЛНИТЬ]` и `[ГОРОД]`/`[ГОД]` на данные своего домена:

1. **Строка 1** — название города и год хроники
2. **Политический ландшафт** — кто контролирует город, какие кланы/секты активны
3. **Историческая канва** — ключевые события истории домена
4. **Атмосфера** — технологии, политика смертных, культура выбранного года
5. **Ключевые локации** — три таблицы (Элизиумы / Серые зоны / Опасные)
6. **Тематические лейтмотивы** — добавь 2–3 темы, специфичных для твоего города
7. **Источники** — укажи книги, актуальные для твоего города (например, *Chicago by Night*, *New York by Night*)

## Шаг 2 — Заполни файлы домена

| Файл | Что заполнить |
|------|--------------|
| `factions.md` | Политическая карта фракций: Князь, Примогены, Анархи, Шабаш |
| `rumors_elysium.md` | 20 слухов для сцен в Элизиуме |
| `rumors_dreaming.md` | 20 слухов Грёзы (если в хронике есть феи; иначе пропусти) |
| `Stories_of_Paris.md` | Переименуй в `Stories_of_[Город].md`, обнови заголовок |
| `rules/timeline.md` | Хронология WoD твоего города (если нужна) |


## Быстрая справка по инструментам

| Команда | Что делает |
|---------|-----------|
| `.\tools\new_npc.ps1 -Name "X" -Type vampire` | Новый канонический НПС |
| `.\tools\new_module.ps1` | Новый модуль (мастер-визард) |
| `.\tools\new_location.ps1 -Name "X" -District "Y" -Zone neutral` | Карточка локации |
| `.\tools\add_diary.ps1 -CharacterName "X" -Period "ГГГГ-ММ"` | Дневниковая запись |
| `.\tools\promote_npc.ps1 -ModuleName "X" -NpcName "Y"` | НПС из модуля → канон |
| `.\tools\validate_links.ps1` | Проверить все ссылки |
| `.\tools\status.ps1` | Статусы персонажей |
| `.\tools\search.ps1 "запрос"` | Поиск по проекту |
