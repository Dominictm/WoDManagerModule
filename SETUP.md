# SETUP — Настройка нового домена VtM

Этот шаблон — готовый фреймворк для ведения Vampire: The Masquerade 20th Anniversary Edition хроники с помощью Claude Code.

---

## Шаг 1 — Запусти мастер настройки

```powershell
.\tools\new_city.ps1
```

Визард спросит:
- Название города (например: Берлин, Москва, Чикаго)
- Год хроники (например: 2010, 1995, 2023)
- Названия районов через запятую (необязательно)

После этого скрипт автоматически:
- Заменит все `[ГОРОД]` и `[ГОД]` в `CLAUDE.md`, `README.md`, `factions.md`, `rumors_*.md`
- Переименует `Stories_of_Paris.md` → `Stories_of_[Город].md`
- Создаст папки районов в `locations/`

Можно также запустить с параметрами напрямую:
```powershell
.\tools\new_city.ps1 -City "Берлин" -Year "2010"
```

---

## Шаг 2 — Заполни CLAUDE.md

Открой `CLAUDE.md` — город и год уже вставлены. Заполни оставшиеся секции `[ЗАПОЛНИТЬ]`:

1. **Политический ландшафт** — кто контролирует город, какие кланы/секты активны
2. **Историческая канва** — ключевые события истории домена для вампиров
3. **Атмосфера** — технологии, политика смертных, культура выбранного года
4. **Ключевые локации** — три таблицы (Элизиумы / Серые зоны / Опасные)
5. **Тематические лейтмотивы** — 2–3 темы, специфичных для твоего города
6. **Источники** — книги по твоему городу (например: *Chicago by Night*, *New York by Night*)

## Шаг 3 — Заполни файлы домена

| Файл | Что заполнить |
|------|--------------|
| `factions.md` | Политическая карта: Князь, Примогены, Анархи, Шабаш |
| `rumors_elysium.md` | 20 слухов для сцен в Элизиуме |
| `rumors_dreaming.md` | 20 слухов Грёзы (если есть феи; иначе пропусти) |
| `rules/open_threads.md` | Открытые сюжетные нити (можно оставить пустым) |

## Справка по инструментам

| Команда | Что делает |
|---------|-----------|
| `.\tools\new_city.ps1` | ★ Настройка нового домена (шаг 1) |
| `.\tools\new_npc.ps1 -Name "X" -Type vampire` | Новый канонический НПС |
| `.\tools\new_module.ps1` | Новый модуль (мастер-визард) |
| `.\tools\new_location.ps1 -Name "X" -District "Y" -Zone neutral` | Карточка локации |
| `.\tools\add_diary.ps1 -CharacterName "X" -Period "ГГГГ-ММ"` | Дневниковая запись |
| `.\tools\promote_npc.ps1 -ModuleName "X" -NpcName "Y"` | НПС из модуля → канон |
| `.\tools\validate_links.ps1` | Проверить все ссылки |
| `.\tools\status.ps1` | Статусы персонажей |
| `.\tools\search.ps1 "запрос"` | Поиск по проекту |
