# Card Schema — карточка персонажа

> Контракт обязательных полей карточки. Проверяется `validate_cards.js`.
> Базовый формат — буллеты `- **Поле:** значение` (не frontmatter), как в `system/rules/npcs_city.md`.

---

## Расположение

| | Текущее | Целевое (после реструктуризации) |
|---|---|---|
| Главная карточка | `characters/<lineage>/<Имя>/<Имя>.md` | `cities/<city>/characters/<lineage>/<slug>/<slug>.md` |
| Лист V20 | `<Имя>/<Имя>-лист.md` | `<slug>/<slug>-sheet.md` |
| Дневники | `<Имя>/Journal_<Имя>/ГГГГ-ММ.md` | `<slug>/journal/{retrospective.md, ГГГГ-ММ.md}` |
| Изображения | `<Имя>/*.png` (рядом) | `<slug>/art/*.png` |

Главная карточка определяется по правилу: имя файла = имя папки (`<slug>/<slug>.md`).
Файлы `-sheet`, папки `journal/`, `art/` — не карточки.

---

## Обязательные элементы

| Элемент | Правило | Пример |
|---|---|---|
| **H1** | `# <эмодзи> <display_name>` | `# 🎪 Верене де Кюстин` |
| **Нав-ссылка** | строка `> 🔗 [Все персонажи](...)` | (+ опц. ссылка на лист) |
| **`Линейка WoD`** | = имя под-папки линейки (см. мэппинг) | `Вампир` → `vampires` |
| **`Статус`** | значение ∈ enum статусов | `Жива` |
| **`## 🖼️ Изображения`** | секция в конце; список арта или `⏳ Изображение не предоставлено` | |

### Новые обязательные поля (вводятся реструктуризацией)

| Поле | Назначение | Пример |
|---|---|---|
| **`Слаг`** | ASCII-ключ папки (`[a-z0-9_]`) | `veren_de_kustin` |
| **`Родной город`** | город-владелец (display) | `Париж` |

### Новые опциональные поля

| Поле | Когда | Пример |
|---|---|---|
| **`Алиасы`** | у персонажа есть вторая личина/кличка | `Кален Урес` |
| **`Присутствие`** | персонаж появлялся вне родного города | `Лос-Анджелес — summer_2011 — 2011-06` |

`display_name` отдельным полем не хранится — это текст H1 без ведущего эмодзи.
Фон/референс изображения (бывш. `npc_image_mapping.md`) переносится в секцию `## 🖼️ Изображения` строкой `<!-- фон: <цвет/описание> -->` или подписью к арту.

## Enum статусов

`Жив`, `Жива`, `Торпор`, `Мёртв`, `Мертва`, `Уничтожен`, `Пропал`, `Неизвестно`

(Модульные неканоничные НПС также: `Активен`.)

---

## Инварианты (проверяет линтер)

1. У каждой папки персонажа есть главная карточка `<slug>/<slug>.md`.
2. Присутствуют H1, нав-ссылка, `Линейка WoD`, `Статус`, секция `## 🖼️ Изображения`.
3. `Статус` ∈ enum.
4. `Линейка WoD` соответствует под-папке линейки.
5. **`Слаг`** уникален в пределах города; **`Алиасы`** не пересекаются с чужими `display_name`/`Слаг`/`Алиасы` в городе.
6. `Родной город` присутствует и указывает на существующий город.

> До Фазы 5 поля `Слаг`/`Родной город` отсутствуют в текущих карточках — линтер по умолчанию выдаёт по ним **WARNING**, а с флагом `--strict` — ERROR (включается после Фазы 5).
