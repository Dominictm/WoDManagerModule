> 📌 **Версия 1.0** | Протокол создания и хранения игромеханических листов персонажей по правилам V20.

---

## 📂 Хранение

Листы хранятся рядом с карточкой персонажа:

```
characters/
└── vampires/
    ├── Верене де Кюстин.md          ← нарративная карточка
    └── Верене де Кюстин-лист.md     ← игромеханический лист (этот протокол)
```

В карточке персонажа добавить строку после заголовка:
```
> 🎲 [Лист персонажа](Имя-лист.md)
```

## 🔁 Протокол создания листа

### Шаг 1 — Концепция
- Концепция (Concept): кем был до обращения
- Клан (Clan)
- Натура (Nature): архетип внутреннего «я»
- Личина (Demeanor): публичная маска
- Поколение (Generation): обычно 8–13 для стартовых ПК
- Сир (Sire)

### Шаг 2 — Атрибуты (все начинают с 1)
Расставить приоритет: **Физические / Социальные / Ментальные**

| Приоритет | Дополнительных точек |
|-----------|---------------------|
| Первичный | 7 |
| Вторичный | 5 |
| Третичный | 3 |

### Шаг 3 — Способности (все начинают с 0, max 3 при создании)
Расставить приоритет: **Таланты / Навыки / Знания**

| Приоритет | Точек |
|-----------|-------|
| Первичный | 13 |
| Вторичный | 9 |
| Третичный | 5 |

### Шаг 4 — Преимущества
- **Дисциплины:** 3 точки (только клановые при создании)
- **Предыстории:** 5 точек
- **Добродетели:** 7 точек (Совесть, Самоконтроль, Мужество; каждая начинает с 1)

### Шаг 5 — Производные характеристики
- **Человечность** = Совесть + Самоконтроль
- **Сила воли** = значение Мужества
- **Пул крови** — по поколению (таблица ниже)

### Шаг 6 — Свободные очки: 15

| Характеристика | Стоимость за точку |
|---|---|
| Атрибут | 5 |
| Способность | 2 |
| Дисциплина | 7 |
| Добродетель | 2 |
| Предыстория | 1 |
| Человечность | 1 |
| Сила воли | 1 |

## ⚠️ Ограничения

- При создании: атрибуты max 5, способности max 3 (без свободных очков)
- Свободные очки не могут поднять способность выше 4 при создании (без одобрения Рассказчика)
- Некланованные дисциплины при создании: только с согласия Рассказчика
- Поколение выше 7-го требует предыстории «Поколение» или Рассказчика
- 📖 Полные правила: [wod.su/vampire](https://wod.su/vampire) · [Дисциплины](https://wod.su/vampire/disciplines)
